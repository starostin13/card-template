# Инструкции по переводу карточек Warhammer 40K

## Правила перевода для AI агентов

### ❌ НЕ ПЕРЕВОДИТЬ:

1. **Названия фаз игры:**
   - Command phase
   - Movement phase
   - Shooting phase
   - Charge phase
   - Fight phase
   - Any phase

2. **Названия шагов:**
   - Battle-shock step
   - Reinforcements step
   - Deep Strike

3. **Слова, написанные ПОЛНОСТЬЮ ЗАГЛАВНЫМИ БУКВАМИ:**
   - INFANTRY
   - VEHICLE
   - CHARACTER
   - WALKER
   - SMOKE
   - GRENADES
   - TITANIC
   - [PRECISION]
   - И любые другие термины в CAPS

4. **Специальные игровые термины:**
   - юнит / unit (оставлять "юнит")
   - модель / model (оставлять "модель")
   - ТРАНСПОРТ / VEHICLE (оставлять "ТРАНСПОРТ" заглавными)
   - ПЕРСОНАЖ / CHARACTER (оставлять "ПЕРСОНАЖ" заглавными)

### ✅ ПЕРЕВОДИТЬ:

- Описания действий и эффектов
- Условия применения (when)
- Целей (target)
- Ограничения (restriction)
- Обычный текст описаний

### Примеры правильного перевода:

**Неправильно:**
```
"Командная фаза, сразу после теста боевого шока для вашего ПЕХОТНОГО юнита"
```

**Правильно:**
```
"Command phase, сразу после теста боевого шока для вашего INFANTRY юнита"
```

**Неправильно:**
```
"Фаза стрельбы, когда модель с ГРАНАТЫ стреляет"
```

**Правильно:**
```
"Shooting phase, когда модель с GRENADES стреляет"
```

**Неправильно:**
```
"Выберите один юнит МАШИНА из вашей армии"
```

**Правильно:**
```
"Выберите один юнит VEHICLE из вашей армии"
```

### Особые случаи:

1. **Benefit of Cover** - переводить как "преимущества укрытия"
2. **Engagement Range** - переводить как "пределы досягаемости"
3. **Hit roll / Wound roll / Damage roll** - "бросок на попадание / ранение / урон"
4. **mortal wound** - "смертельная рана"
5. **invulnerable save** - "непробиваемое спасение"

### Структура карточки:

```json
{
  "title": "НАЗВАНИЕ НА АНГЛИЙСКОМ",  // НЕ переводить
  "body": {
    "when": "Название фазы + русское описание",
    "target": "Игровые термины + русское описание",
    "effect": "Игровые термины + русское описание",
    "restriction": "Полностью на русском"
  }
}
```

### Контрольный список перед переводом:

- [ ] Фазы оставлены на английском?
- [ ] ЗАГЛАВНЫЕ термины не переведены?
- [ ] "юнит", "модель" используются вместо "unit", "model"?
- [ ] VEHICLE → ТРАНСПОРТ (заглавными)?
- [ ] CHARACTER → ПЕРСОНАЖ (заглавными)?
- [ ] INFANTRY, SMOKE, GRENADES и др. не переведены?
- [ ] Заголовок карточки (title) остался на английском?
